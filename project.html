Command-Line Q&A Model Fine-Tuning

This project showcases fine-tuning of small open-weight language models (TinyLlama-1.1B, Phi-2, GPT-Neo-125M) using LoRA on 150+ command-line Q&A pairs covering Git, Bash, tar/gzip, grep, and venv topics. It features a Python-based CLI agent that accepts natural-language tasks, generates clear step-by-step plans, and safely dry-runs shell commands (echo ), with logs stored in JSONL format. The fine-tuning and evaluation (BLEU, ROUGE-L, plan quality) were done on a free Google Colab T4 GPU. Tools used include LoRA, Python, Colab, and standard NLP evaluation libraries.
